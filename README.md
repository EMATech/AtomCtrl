# AtomCtrl
 Presonus ATOM Controller

## LoopMIDI
Create an AtomCtrl port
Configure your DAW to use this port rather than the ATOM port
```
 +------+                +----------+                +-----+
 | ATOM | <- USB MIDI -> | AtomCtrl | <- loopMIDI -> | DAW |
 +------+                +----------+                +-----+
```
